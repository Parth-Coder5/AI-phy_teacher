# AI Physics Teacher

A Retrieval-Augmented Generation (RAG) AI system that answers physics questions using a custom knowledge base.

Built with:
- Python
- Sentence Transformers
- Vector Search
- Local LLM (Ollama / Mistral)

Features:
- Physics knowledge base
- Embedding search
- Similarity filtering
- Context-based answers

Run:

python final_tutor.py
